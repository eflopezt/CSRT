Ayuda
¿Qué hace la sincronización?

Esta herramienta automatiza la creación y vinculación de usuarios del sistema con los registros de personal.

Vincular Existentes:

Busca usuarios existentes cuyo username coincida con el DNI del personal y los vincula automáticamente.

Crear Nuevos:

Crea nuevos usuarios para el personal que no tiene usuario asignado. El username será la primera letra del nombre + apellido paterno (ej: Juan Pérez → jperez). La contraseña será el DNI.

Importante: Esta operación no es reversible automáticamente. Los usuarios creados permanecerán en el sistema.
Comando de Terminal:
python manage.py sincronizar_usuarios --help
